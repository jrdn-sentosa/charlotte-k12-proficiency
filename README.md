# Charlotte K-12 Proficiency Predictor

Predicting K-12 test proficiency across Charlotte-Mecklenburg neighborhoods using environmental and socioeconomic factors from the [Quality of Life Explorer](https://maps.mecklenburgcountync.gov/qol/#20/).

## Overview

This project builds regression and classification models to identify neighborhood-level drivers of academic outcomes across 455 NPAs (Neighborhood Profile Areas) in Charlotte-Mecklenburg County. We found that access to food and nutrition services and household income are the strongest predictors of test proficiency.

## Key Results

- **Primary driver:** Food and Nutrition Services
- **Secondary driver:** Household Income
- **Best regression model:** Random Forest
- **Best classification model:** XGBoost
